# Set Inclusion

<details>

<summary><span class="math">\textbf{Includes} \; A \; B</span></summary>

***

$$A \in \textbf{Set}$$

$$B \in \textbf{Set}$$

$$\forall (x : x \in A : x \in B)$$

***

**Notation**

1. $$A \subseteq B$$ denotes $$\textbf{Includes} \; A \; B$$.

***

```
pred Includes(A,B: set univ) {
  A in B
}
```

</details>
